## Projeto CEC Cells 

Repositório utilizado para compartilhamento de análises e artigos do Projeto CEC Cells CNPq.


## Organização do Repositório

```
├── analysis       <- Principais códigos utilizados para realização de experimentos, análises e/ou testes;
├── data           <- Diretório com as principais fontes de dados do projeto.
│   └── *.csv      <- Conjuntos/fontes de dados utilizadas no projeto.
├── papers
│   └── *.pdf      <- Artigos com temas relacionados ao projeto no geral.
├── README.md      <- Arquivo com as principais informações do projeto/código.
```
